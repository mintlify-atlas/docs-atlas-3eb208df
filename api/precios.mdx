Returns the most recent price data for all 10 cryptocurrencies tracked by CryptoTracker Web.

## Endpoint

```
GET http://your-server:8080/precios
```

## Parameters

This endpoint does not accept any parameters.

## Response

### Response Format

Returns a JSON array containing price objects for all cryptocurrencies:

<ResponseField name="root" type="array">
  Array of cryptocurrency price objects
  
  <ResponseField name="name" type="string" required>
    Full name of the cryptocurrency
  </ResponseField>
  
  <ResponseField name="symbol" type="string" required>
    Trading symbol of the cryptocurrency
  </ResponseField>
  
  <ResponseField name="price" type="number" required>
    Current price in USD (formatted with 2 decimal places)
  </ResponseField>
</ResponseField>

## Implementation Details

### Data Source

The endpoint queries a MySQL database to retrieve the most recent price record for each cryptocurrency. Each cryptocurrency has its own table:

- `bitcoin`, `ethereum`, `ripple`, `solana`, `tron`
- `dogecoin`, `cardano`, `hyperliquid`, `bitcoin_cash`, `chainlink`

### Query Logic

For each cryptocurrency table, the API executes:
```sql
SELECT precio FROM {table_name} 
ORDER BY fecha_registro DESC 
LIMIT 1
```

### Price Updates

Prices are automatically updated every 60 seconds by the collector service, which fetches data from the CoinGecko API.

## Example Requests

### cURL

```bash
curl http://your-server:8080/precios
```

### JavaScript (Fetch)

```javascript
fetch('http://your-server:8080/precios')
  .then(response => response.json())
  .then(prices => {
    prices.forEach(crypto => {
      console.log(`${crypto.name} (${crypto.symbol}): $${crypto.price}`);
    });
  })
  .catch(error => console.error('Error:', error));
```

### Python (requests)

```python
import requests

response = requests.get('http://your-server:8080/precios')
prices = response.json()

for crypto in prices:
    print(f"{crypto['name']} ({crypto['symbol']}): ${crypto['price']}")
```

### Java

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class CryptoPrice {
    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("http://your-server:8080/precios"))
            .GET()
            .build();
        
        HttpResponse<String> response = client.send(request, 
            HttpResponse.BodyHandlers.ofString());
        
        System.out.println(response.body());
    }
}
```

## CORS Support

The endpoint includes full CORS support:

**Response Headers:**
```
Content-Type: application/json
Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, POST, OPTIONS
Access-Control-Allow-Headers: *
```

**OPTIONS Preflight:**
```bash
curl -X OPTIONS http://your-server:8080/precios
# Returns 204 No Content with CORS headers
```

## Notes

- All prices are in USD
- Prices are formatted to 2 decimal places
- The endpoint always returns data for all 10 cryptocurrencies (if available in the database)
- If a cryptocurrency has no data in the database, it will be omitted from the response
- The response is manually constructed as JSON (not using a JSON library)
