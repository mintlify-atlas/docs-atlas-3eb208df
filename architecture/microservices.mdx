## Microservices Architecture

CryptoTracker Web consists of three independent Java microservices, each running on its own port and serving a specific purpose in the distributed system.

## Service Overview

| Service | Port | Purpose | Dependencies |
|---------|------|---------|-------------|
| WebServer | 8080 | REST API for price data | MySQL |
| GraficoService | 8081 | Chart generation and visualization | MySQL, XChart |
| RecolectorPrecios | 8082 | Price collection and persistence | MySQL, CoinGecko API |

#### GET /grafico
Generates a price variation chart for a single cryptocurrency.

**Parameters:**
- `crypto` (required) - Cryptocurrency name (e.g., "bitcoin")
- `horas` (optional) - Number of hours to display (1-24, default: 3)

**Example:**
```
http://localhost:8081/grafico?crypto=bitcoin&horas=6
```

**Response:** PNG image (800x450px)

**Implementation:**
```java
// GraficoService.java:79-128
private static void handleGraficoRequest(HttpExchange exchange) throws IOException {
    setCORSHeaders(exchange);
    
    if ("OPTIONS".equalsIgnoreCase(exchange.getRequestMethod())) {
        handleOptionsRequest(exchange);
        return;
    }
    
    Map<String, String> params = parseQuery(exchange.getRequestURI().getQuery());
    String crypto = params.getOrDefault("crypto", "").toLowerCase();
    int horas = Integer.parseInt(params.getOrDefault("horas", "3"));
    
    if (crypto.isEmpty() || horas <= 0 || horas > 24) {
        String error = "Parámetros inválidos. Usa ?crypto=bitcoin&horas=3";
        exchange.sendResponseHeaders(400, error.length());
        return;
    }
    
    try {
        List<Registro> registros = obtenerDatos(crypto, horas);
        if (registros.isEmpty()) {
            exchange.sendResponseHeaders(204, -1);
            return;
        }
        
        byte[] imageBytes = generarGraficoPNG(crypto, registros);
        exchange.getResponseHeaders().set("Content-Type", "image/png");
        exchange.sendResponseHeaders(200, imageBytes.length);
        try (OutputStream os = exchange.getResponseBody()) {
            os.write(imageBytes);
        }
    } catch (SQLException e) {
        // Error handling
    }
}
```

private static byte[] generarGraficosTodasCriptos(int horas) throws SQLException, IOException {
    final int graficosAncho = 400;
    final int graficosAlto = 300;
    final int filas = 5;
    final int columnas = 2;
    final int margen = 20;
    
    int imagenAncho = columnas * graficosAncho + (columnas + 1) * margen;
    int imagenAlto = filas * graficosAlto + (filas + 1) * margen + 50;
    
    BufferedImage imagenCompuesta = new BufferedImage(imagenAncho, imagenAlto, BufferedImage.TYPE_INT_RGB);
    Graphics2D g2d = imagenCompuesta.createGraphics();
    
    // Draw background and title
    g2d.setColor(Color.WHITE);
    g2d.fillRect(0, 0, imagenAncho, imagenAlto);
    
    // Generate individual charts
    for (int i = 0; i < criptosOrdenadas.size(); i++) {
        String crypto = criptosOrdenadas.get(i);
        List<Registro> registros = obtenerDatos(crypto, horas);
        
        XYChart chart = crearGraficoIndividual(crypto, registros, graficosAncho, graficosAlto);
        BufferedImage graficoImg = BitmapEncoder.getBufferedImage(chart);
        
        int fila = i / columnas;
        int columna = i % columnas;
        int x = margen + columna * (graficosAncho + margen);
        int y = 60 + margen + fila * (graficosAlto + margen);
        
        g2d.drawImage(graficoImg, x, y, null);
    }
    
    ByteArrayOutputStream out = new ByteArrayOutputStream();
    ImageIO.write(imagenCompuesta, "PNG", out);
    return out.toByteArray();

#### GET /regresion
Generates linear regression analysis for a cryptocurrency within a time range.

**Parameters:**
- `crypto` (required) - Cryptocurrency name
- `inicio` (optional) - Start hour (0-23, default: 10)
- `fin` (optional) - End hour (0-24, default: 12)

**Example:**
```
http://localhost:8081/regresion?crypto=bitcoin&inicio=10&fin=14
```

**Response:** PNG image with scatter plot and regression line

**Features:**
- Displays actual data points as scatter plot
- Overlays linear regression line
- Shows equation (y = mx + b) and R² coefficient

**Regression Calculation:**
```java
// GraficoService.java:718-745
private static RegresionLineal calcularRegresionLineal(List<Double> x, List<Double> y) {
    int n = x.size();
    double sumX = 0, sumY = 0, sumXY = 0, sumX2 = 0, sumY2 = 0;
    
    for (int i = 0; i < n; i++) {
        sumX += x.get(i);
        sumY += y.get(i);
        sumXY += x.get(i) * y.get(i);
        sumX2 += x.get(i) * x.get(i);
        sumY2 += y.get(i) * y.get(i);
    }
    
    double pendiente = (n * sumXY - sumX * sumY) / (n * sumX2 - sumX * sumX);
    double interseccion = (sumY - pendiente * sumX) / n;
    
    double mediaY = sumY / n;
    double ssTot = 0, ssRes = 0;
    
    for (int i = 0; i < n; i++) {
        double yPred = pendiente * x.get(i) + interseccion;
        ssTot += Math.pow(y.get(i) - mediaY, 2);
        ssRes += Math.pow(y.get(i) - yPred, 2);
    }
    
    double r2 = 1 - (ssRes / ssTot);
    
    return new RegresionLineal(pendiente, interseccion, r2);
}
```

### Data Retrieval

```java
// GraficoService.java:130-158
private static List<Registro> obtenerDatos(String crypto, int horas) throws SQLException {
    List<Registro> datos = new ArrayList<>();
    
    try {
        Class.forName("com.mysql.cj.jdbc.Driver");
    } catch (ClassNotFoundException e) {
        throw new SQLException("Driver JDBC no encontrado");
    }
    
    try (Connection conn = DriverManager.getConnection(DB_URL)) {
        String query = String.format("""
            SELECT fecha_registro, precio FROM %s
            WHERE fecha_registro >= NOW() - INTERVAL ? HOUR
            ORDER BY fecha_registro
        """, crypto);
        
        try (PreparedStatement stmt = conn.prepareStatement(query)) {
            stmt.setInt(1, horas);
            ResultSet rs = stmt.executeQuery();
            while (rs.next()) {
                LocalDateTime fecha = rs.getTimestamp("fecha_registro").toLocalDateTime().minusHours(6);
                double precio = rs.getDouble("precio");
                datos.add(new Registro(fecha, precio));
            }
        }
    }
    
    return datos;
}
```

### Scheduled Task

#### TareaObtenerPrecios (Inner Class)

This `TimerTask` runs every 60 seconds to fetch and store prices:

```java
// RecolectorPrecios.java:92-109
static class TareaObtenerPrecios extends TimerTask {
    private final HttpClient cliente = HttpClient.newHttpClient();
    
    @Override
    public void run() {
        HttpRequest solicitud = HttpRequest.newBuilder()
            .uri(URI.create(URL_API))
            .GET()
            .build();
        
        cliente.sendAsync(solicitud, HttpResponse.BodyHandlers.ofString())
            .thenApply(HttpResponse::body)
            .thenAccept(this::transformarYGuardarPrecios)
            .exceptionally(error -> {
                System.err.println("Error al obtener precios: " + error.getMessage());
                return null;
            });
    }
}
```

### Data Processing

#### JSON Parsing and Database Insertion

```java
// RecolectorPrecios.java:111-144
private void transformarYGuardarPrecios(String jsonOriginal) {
    try {
        String[] criptos = {"bitcoin", "ethereum", "ripple", "solana", "tron", 
                           "dogecoin", "cardano", "hyperliquid", "bitcoin_cash", "chainlink"};
        
        // Remove outer braces
        jsonOriginal = jsonOriginal.substring(1, jsonOriginal.length() - 1);
        
        // Split by commas followed by quotes
        String[] monedas = jsonOriginal.split(",(?=\\\"[a-z])");
        
        try (Connection conexion = DriverManager.getConnection(DB_URL)) {
            for (int i = 0; i < monedas.length; i++) {
                String moneda = monedas[i].trim();
                String precioStr = moneda.split(":")[2].replace("}", "").trim();
                double precio = Double.parseDouble(precioStr);
                
                // Insert into database
                String tabla = criptos[i];
                String sql = "INSERT INTO " + tabla + " (precio) VALUES (?)";
                
                try (PreparedStatement pstmt = conexion.prepareStatement(sql)) {
                    pstmt.setDouble(1, precio);
                    pstmt.executeUpdate();
                    System.out.println(tabla + ": precio=" + precio);
                }
            }
        }
    } catch (Exception e) {
        System.err.println("Error al procesar datos: " + e.getMessage());
    }
}
```

## Inter-Service Communication

### Communication Pattern

The services use a **shared database** pattern rather than direct HTTP communication:

```
RecolectorPrecios ──▶ [MySQL] ◀── WebServer
                         ▲
                         │
                         └──── GraficoService
```

**Benefits:**
- Simple architecture
- No service-to-service dependencies
- Natural data persistence
- Easy to add new consumers

**Drawbacks:**
- Database becomes a bottleneck
- Tight coupling through schema
- No real-time updates (polling required)

### JDBC Configuration

All services use the same connection string:
```java
private static final String DB_URL = 
    "jdbc:mysql://10.23.176.2:3306/criptomonedas_db?user=root&password=root&useSSL=true";
```

### Connection Management

**Pattern:** Connection-per-request with try-with-resources

```java
try (Connection conexion = DriverManager.getConnection(DB_URL)) {
    // Database operations
} // Automatic connection close
```

## Performance Characteristics

### WebServer
- **Thread Pool:** 8 threads
- **Response Time:** < 100ms (database query)
- **Concurrency:** Can handle 8 simultaneous requests
- **Bottleneck:** Database query performance

### GraficoService
- **Thread Pool:** 4 threads
- **Response Time:** 500ms - 2s (depending on chart complexity)
- **Concurrency:** 4 simultaneous chart generations
- **Bottleneck:** Chart rendering (CPU-intensive)
- **Memory:** High memory usage for image generation

### RecolectorPrecios
- **Execution Interval:** 60 seconds
- **API Call:** Asynchronous with HttpClient
- **Database Writes:** 10 INSERT operations per cycle
- **Bottleneck:** Network latency to CoinGecko API
