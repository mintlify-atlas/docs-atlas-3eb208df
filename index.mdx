  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with CryptoTracker Web in minutes</p>
  
  <Steps>
    <Step title="Clone the repository">
      Clone the CryptoTracker Web repository to your local machine.
      
      ```bash
      git clone https://github.com/Damrod14/CryptoTracker_Web.git
      cd CryptoTracker_Web
      ```
    </Step>
    
    <Step title="Configure the database">
      Set up MySQL and create the required database schema. Update the connection string in each service.
      
      ```java
      private static final String DB_URL = 
        "jdbc:mysql://localhost:3306/criptomonedas_db?user=root&password=your_password";
      ```
      
      <Note>Each cryptocurrency has its own table: `bitcoin`, `ethereum`, `ripple`, `solana`, `tron`, `dogecoin`, `cardano`, `hyperliquid`, `bitcoin_cash`, and `chainlink`.</Note>
    </Step>
    
    <Step title="Start the microservices">
      Compile and run each service on its designated port.
      
      ```bash
      # WebServer (port 8080)
      javac -cp ".;lib/*" java/WebServer.java
      java -cp ".;lib/*" WebServer
      
      # GraficoService (port 8081)
      javac -cp ".;lib/*" java/GraficoService.java
      java -cp ".;lib/*" java.GraficoService
      
      # RecolectorPrecios (port 8082)
      javac -cp ".;lib/*" java/RecolectorPrecios.java
      java -cp ".;lib/*" RecolectorPrecios
      ```
    </Step>
    
    <Step title="Access the frontend">
      Open the frontend in your browser to start tracking cryptocurrencies.
      
      ```
      http://localhost/index.html
      ```
      
      The interface provides real-time price updates, historical charts, comparison tools, and linear regression analysis.
    </Step>
  </Steps>
</div>
